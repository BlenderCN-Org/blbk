使用方法
---------
ここからg_toolsフォルダーをdlして、ブレンダーのaddonsフォルダーに入れる。<br>
例えば<br>
C:\Program Files\Blender Foundation\Blender 2.79\2.79\scripts\addons <br>
にぶち込むだけでいいはず<br>
後は祈り
